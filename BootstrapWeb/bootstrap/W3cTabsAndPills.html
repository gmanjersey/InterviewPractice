Toggleable / Dynamic Tabs

To make the tabs toggleable, add the data-toggle="tab" attribute to each link. Then add a .tab-pane class with a unique ID for every tab and wrap them inside a

element with class .tab-content. If you want the tabs to fade in and out when clicking on them, add the .fade class to .tab-pane:
